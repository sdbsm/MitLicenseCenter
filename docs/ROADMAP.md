## Активный трек — Редизайн UX/UI (поставлен 2026-06-18)

Полный UX/UI-редизайн панели поверх существующего стека (уже shadcn/ui + Radix + Tailwind v4 —
это дизайн-работа постранично, **не** миграция), под прикрытием FE-тестов, по PR на задачу.
Решения согласованы с владельцем в чате-консультации; полная спека (per-screen + риски) — в
локальном план-файле `.claude/plans/redesign-ux.md`, активная задача — в
[`PROJECT_BACKLOG.md`](PROJECT_BACKLOG.md).

**Зафиксировано:**
- **Палитра — строгий монохром:** нейтральные `oklch`-токены `index.css` + цвет только через
  `StatusBadge` (статусная семантика); **тёмная тема — основная**; бренд-акцент отклонён владельцем.
- **IA: меню 8 → 7.** «Сеансы» становятся **домом темы лицензий** (проекции «По клиентам» /
  «Живые сеансы»); **«Отчёты» растворяются** (тренд лицензий → «Сеансы», размер баз → «Базы»).
  Двигает реальные фичи/контракты → отдельные ADR (навигация, растворение «Отчётов») + parity BE↔FE.
- **«Быстродействие»** переосмысляется в **воронку расследования** (вердикт → светофор ресурсов →
  атрибуция семей → drill-down Хост/1С/SQL с авто-фокусом слоя → единый блок «блокировки»): это
  изменение логики, не только стиль (FE-склейка + маленькая производная на BE).

**Фазы:** 0 Дизайн-система (риск низкий) → 1 Навигация/IA (высокий) → 2 Экраны по одному (средний)
→ 3 «Быстродействие» (высокий). Старт — `MLC-195` (Фаза 0).

> **Принятые «нет/потом» здесь не дублируются** — это решения, а не планы вперёд, их дом —
> [`DECISIONS.md`](DECISIONS.md): multi-cluster/multi-node gated за **ADR-28** (single-host);
> RAS Strategy B отклонён (**ADR-3.3**); планировщик резервного копирования, restore-оркестрация
> и встроенная 2FA — вне сферы навсегда (**ADR-15**; единственное исключение — ручной on-demand
> `COPY_ONLY`-бэкап, **ADR-27**).
