### Активный трек — «Сервер» (релиз 1.1.3)

Управление службами узла + мониторинг обслуживания: новый раздел меню **«Сервер»** с тремя
вкладками («Службы» / «IIS» / «Обслуживание») — статус и жизненный цикл служб стека (сервер 1С,
RAS, IIS; **SQL — только наблюдение**), рабочие процессы 1С (`rphost`), расписание авто-рестартов
сервера 1С, мониторинг свежести бэкапов и планов обслуживания SQL (только чтение, с детализацией по
задачам). Управление IIS **переезжает** из вкладки в «Базах» сюда. Надёжность операций со службами —
команда + верификация фактического состояния с таймаутом (**ADR-55**); экран и информационная
архитектура — **ADR-54**. Постановка 2026-06-19, цепочка `MLC-212..219` — активный реестр в
[`PROJECT_BACKLOG.md`](PROJECT_BACKLOG.md). Закрыто 2026-06-19: ✅ `MLC-212` (надёжный контроллер
службы Windows, ADR-55) + ✅ `MLC-213` (агрегатор статусов служб `/api/v1/server` + управление
сервером 1С) + ✅ `MLC-214` (FE-экран «Сервер» + вкладка «Службы» + индикатор здоровья на «Обзоре»);
в работе `MLC-215` (переезд управления IIS в вкладку «IIS» нового экрана).

Прежние роадмап-планы v1.1.x выпущены (`v1.1.0` / `1.1.1` / `1.1.2`). Прочие кандидаты вперёд —
отложенные опции в [`PROJECT_BACKLOG.md`](PROJECT_BACKLOG.md) (раздел «Открытые опции»).
