# MitLicense Center — Roadmap

Документ описывает актуальные планы развития и отложенный бэклог.
Горизонт — пострелизный цикл (v1.x и далее). Продуктовый контекст —
`01_OVERVIEW.md`; архитектурные ограничения — `DECISIONS.md`.

## Релиз `0.5.1-beta` — выпущен (2026-06-15)

Минор по запросу владельца: трек «Свежесть данных на страницах» (`MLC-154..156`). Устранена
задержка появления данных — холодный обход сеансов вынесен из Hangfire в
`ColdTierPollingService : BackgroundService`, настройка `Polling.ColdIntervalSeconds` стала
реальной (60с → дефолт 15с, ADR-6.1 Update); админ-списки освежаются при возврате на вкладку;
на «Сеансах»/«Быстродействии» — `LiveControls` (Пауза + «Обновить сейчас», на «Сеансах» —
живой форс-обход 1С). Индекс — `PROJECT_BACKLOG_ARCHIVE.md` («Свежесть данных на страницах»).

- `MLC-154` — cold-цикл → `BackgroundService`, реальный каданс (главный эффект).
- `MLC-155` — `refetchOnWindowFocus: true` (застой админ-списков).
- `MLC-156` — `LiveControls`: Пауза + «Обновить сейчас» (live-форс через `POST /sessions/refresh`).

## Релиз `0.5.0-beta` — выпущен (2026-06-14)

Минор по итогам приёмки `0.4.0-beta` + запросы владельца, одним релизом. Все 7 задач закрыты,
версия забампана `0.5.0-beta`. Индекс — `PROJECT_BACKLOG_ARCHIVE.md` («Релиз 0.5.0-beta»).

- `MLC-149` — версия панели в подвале сайдбара (источник — рантайм `/api/v1/health`).
- `MLC-148` — фикс холодного старта `/sessions` («~2000 лет назад»): прогрев + защита `RelativeTime`.
- `MLC-147` — xlsx (SheetJS) → CDN-`0.20.3` (обе high закрыты).
- `MLC-027` — `i18n/ru.json` разнесён на per-feature файлы (один namespace, call-sites не тронуты).
- `MLC-150` — серверный фильтр «Базы» по «не найдена в кластере».
- `MLC-151` — optimistic concurrency на Infobase/Publication (rowversion→409).
- `MLC-152`/`MLC-153` — low-priv аккаунт службы: трасса проведена, но переход **не предпринят** и
  кодовая попытка снять `sysadmin` с бэкапа **откачена** (`MLC-153`). Вывод: для мультитенанта с
  динамическими базами `sysadmin`/admin — штатный профиль (бэкап клиентских баз требует прав per-DB,
  а серверного «backup any DB» короче `sysadmin` нет; IIS-функции по природе требуют admin). См. ADR-44.

> **Живая проверка на стенде (за владельцем):** холодный старт `/sessions` после рестарта (MLC-148).

## Актуальные планы (вне релиза 0.4.0-beta)

### 1. ESLint-правило для StatusBadge

`StatusBadge` — единственный способ отображать статусы в UI; правило его применения
поддерживается ревью, не линтером. Запланировано: кастомное ESLint-правило,
форсящее `StatusBadge` вместо сырых цветовых классов.

### 2. RAS Strategy B — **отклонено** (2026-06-14)

Текущее состояние: `rac.exe` запускается как отдельный процесс на каждый цикл.
Кросс-вызовой кэш UUID кластера уже сократил steady-state spawn rate примерно
вдвое; бюджет ≤ 26 proc/min выдерживается с запасом.

«Долгоживущий TCP-сокет на порту 1545» (Strategy B) означал бы **переписать закрытый
бинарный протокол RAS на .NET** (реверс-инжиниринг, ломкость по версиям 8.5.x, потеря
страховки официального `rac.exe`). Измеренной проблемы латентности нет (бюджет держится
с запасом), поэтому решением владельца (2026-06-14) **отклонено**. Регистрация `ras.exe`
службой Windows — ортогональна (серверная сторона), не замена Strategy B. Официальный
`rac.exe` остаётся единственным адаптером кластера (ADR-16). Вернуть можно только при
измеренной проблеме латентности на боевой нагрузке.

### 3. Multi-cluster / multi-node топология

Все адаптеры предполагают single-node (зафиксировано ADR-28). Расширение требует
полного пересмотра каждого адаптера и операционной модели. Брать только при явном
запросе со стороны оператора и после отзыва ADR-28.

> Бывшие пункты «Серверный фильтр «Базы»», «Optimistic concurrency на Infobase/Publication» и
> «Low-priv аккаунт службы» взяты в трек `0.5.0-beta` (`MLC-150`/`MLC-151`/`MLC-152`).

## Permanently out of scope (ADR-15)

Планировщик резервного копирования, restore-оркестрация внутри панели и встроенная
2FA. Исключение: on-demand `COPY_ONLY`-бэкап (ADR-27). Вернуть любой пункт можно
только явным отзывом ADR-15.
